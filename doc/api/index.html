src/construct.js

Constructs a new object based on its class name that inherits from a specific class.

construct(parentClass: Object, configuration: Object, logbook: Logbook, restricted: boolean)
Parameters
parentClass (Object) The class object of the class to be inherited
configuration (Object) The configuration for the object that is created
Name Description
configuration.module string The JavaScript module that contains the class to be instantiated
configuration.class string The name of the class (inside the module) to be instantiated - the constructor must take the sub-configuration and the logbook as its two only parameters
configuration.configuration Object The sub-configuration that is passed to the constructor
logbook (Logbook) The logbook to log
restricted (boolean = true) Whether the module can be a full URL (allowing to construct classes from other sources)
