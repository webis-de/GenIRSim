# GenIRSim

Quickstart:
```
npm install
npm exec genirsim configurations/default-configuration.json > eval.json
```

To run [web server](http://localhost:8000):
```
npm install
npm exec genirsim-server
```


## Maintenance
```
npm run doc
docker build -t ghcr.io/webis-de/genirsim:$(jq -r '.version' package.json) .
```
